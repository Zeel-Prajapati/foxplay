Tower Building

The game Tower Building start with three hearts, show in the top-right corner. Player loses one heart whenever tower block falls, and the game end if all heart are gone. Scoring is works by stack tower block normal stack give 25 points while Perfect stack give 50 points and if perfect block align in continous give 50 and extra 25 points as a bonus.
